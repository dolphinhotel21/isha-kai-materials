## 📋 資料の内容

要支援・軽度者への支援の流れと、医師会へのお願いをまとめた1枚もの説明資料。

- **背景**：2040年問題（介護人材不足）＋ 2024年度ガイドライン改定
- **現場の課題**：サービス固定化・資源配分の偏り・入口の問題
- **市の方針**：リエイブルメント推進
- **支援の流れ**：患者 → 医療機関 → 地域包括支援センター → 振り分け
- **医師へのお願い**：①相談先案内 ②専門的助言 ③意見書作成
